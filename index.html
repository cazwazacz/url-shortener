URL Shortener Microservice


Append a full URL to the /new directory of the root URL to retrieve a JSON object with a shortened version of it.

Visiting the shortened URL redirects you to the original page


For example, copying this into the address bar:

https://www.caz-urlshortener.com/new/http://www.google.com/

Should give you the URL:

https://caz-urlshortener.com/B1w0tNqwg
